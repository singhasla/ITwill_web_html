padding - 요소의 내부여백

padding값의 갯수별 적용

  • padding: n; 전방향
  • padding: n n; 상하, 좌우
  • padding: n n n; 상, 좌우, 하
  • padding: n n n n; 상, 우, 하, 좌

polaroid (padding) 효과

padding의 방향별 적용 속성

  • padding-top: n; 위쪽에만 내부여백 n이 적용됨.
  • padding-right: n; 오른쪽에만 내부여백 n이 적용됨.
  • padding-bottom: n; 아래쪽에만 내부여백 n이 적용됨.
  • padding-left: n; 왼쪽에만 내부여백 n이 적용됨.
