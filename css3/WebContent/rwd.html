미디어 쿼리 구문

@media only|not 장치종류 and ( 조건 ) { css선택자 { 속성: 속성값 ;} } * only나 not은 필요한 경우에만 사용

장치종류

  • all - 모든 디바이스
  • print - 프린터
  • screen - screen형태로 나타나는 모든 computer (tablets, smart-phones etc.)
  • speech - 음성낭독기

기타 다른 조건들...

  • width and height of the viewport - 뷰포트의 너비와 높이
  • width and height of the device - 장치의 너비와 높이
  • orientation (landscape or portrait) - 가로 또는 세로 방향
  • resolution - 해상도
