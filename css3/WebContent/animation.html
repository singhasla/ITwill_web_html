2. animation속성으로 정의한 애니메이션을 요소에 적용하기

animation의 하위속성들

  1. animation-name: ; 애니메이션 이름
  2. animation-duration:ns/nms; 애니메이션 진행시간. 단위는 초나 밀리초
  3. animation-delay:ns/nms; 애니메이션 지연시간. 단위는 초나 밀리초
  4. animation-iteration-count:n; 애니메이션 반복횟수 (무한반복은 infinite)
  5. animation-direction: ; 애니메이션 재생방향
    • normal-애니메이션이 정상적으로 (앞으로) 재생됩니다. 기본값 0%~100%
    • reverse -애니메이션이 역방향 (역방향)으로 재생됩니다. 100%~0%
    • alternate -애니메이션이 먼저 정방향 재생 된 다음 역방향으로 재생됩니다. 0%~100%~0%
    • alternate-reverse -애니메이션이 먼저 역방향 재생 된 다음 정방향으로 재생됩니다. 100%~0%~100%
  6. animation-timing-function 애니메이션 가속감속 등
    • ease (기본값 ) = 느리게 시작하고 빨라지다가 천천히 종료하는 효과
    • ease-in = 느리게 시작하는 효과
    • ease-out = 느리게 끝나는 효과
    • ease-in-out = 느리게 시작하고 느리게 끝나는효과
    • linear = 동일한 속도
    • cubic-bezier(n,n,n,n) = cubic-bezier 함수값으로 적용
      https://cubic-bezier.com/
  7. animation-fill-mode 애니메이션 멈춤시의 장면 지정(다있소 참조)
  8. animation-play-state:running/paused; 애니메이션 실행/일시 중지
