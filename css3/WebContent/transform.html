transform - 변형

변형 함수들

변형 함수 2개 이상 적용

translate rotate
rotate translate
  • transform:translate(Xn,Yn); 위치를 X방향으로 n만큼, Y방향으로 n만큼 변형. n은 음수값도 가능. (단위는 px/%/em/등등...)
  • transform:scaleX(n); 크기를 X방향으로 변형. (n은 1보다 작으면 축소, 크면 확대 됨)
  • transform:scaleY(n); 크기를 Y방향으로 변형.
  • transform:scale(n,n); 크기를 X방향 Y방향으로 동시에 변형.
  • transform:rotateX(n); 가로축을 기준으로 회전 변형. (단위는 deg, 음수값 가능)
    transform:rotateY(n); 세로축을 기준으로 회전 변형.
    transform:rotate(n); (바람개비처럼 정면에서 보이는) 회전 변형.
  • transform:skewX(n); 가로축을 기준으로 기울기 변형. (단위는 deg, 음수값 가능)
    transform:skewY(n); 세로축을 기준으로 기울기 변형.
    transform:skew(n,n); 가로,세로축을 기준으로 동시에 기울기 변형.

transform-origin: n,n ; 변형의 기준점 변경 (기본값은 정중앙임)

가로 세로의 순서로 적용.
방향값을 조합하거나 - left/center/right top/center/bottom
다양한 단위를 붙여 직접값을 지정 - npx npx / n% n% / nem nem / ...

transform:translate(); 위치 변형

transform:translate(800px,0);
transform:translate(0,800px);
transform:translate(-200px,0);
transform:translate(0,-400px);
transform:translate(900px,-300px);
transform:translate(40em,10em);
transform:translate(100%,100%);
transform:translate( ); 단위가 %일 경우 자신의 크기가 기준.
항상 부모의 center에 위치하기 [부모의 설정] position: relative; [자식의 설정] position: absolute; left: 50%; top:50%; - 부모의 정중앙에서 자식의 시작점이 위치함 (크기가 클수록 중앙에서 벗어남) transform:translate(-50%,-50%); - 자식의(자신의) 크기 절반만큼 위와 앞으로 이동시켜 정중앙을 위치를 완성.

요소에 ★ box-sizing: border-box; ★를 지정하면 padding값과 border를 추가해도 요소의 크기가 더이상 커지지 않는다. ( 기본값 content-box는 width와 height안에 padding과 border를 포함하지 않음)

transform:scale(); 크기 변형

transform:scaleX(0.5);
transform:scaleX(2);
transform:scaleY(0.4);
transform:scaleY(2.4);
transform:scale(2,2);
transform:scale(2,2.5);

transform:rotate(); 회전 변형

transform:rotateX(90deg);
transform:rotateY(90deg);
transform:rotateX(-60deg);
transform:rotateY(-180deg);
transform:rotate(360deg);
transform:rotate(-460deg);
transform:rotateX(-720deg);

transform:skew();

transform:skewX(20deg);
transform:skewX(-20deg);
transform:skewY(40deg);
transform:skewY(-40deg);
transform:skew(30deg,30deg);
transform:skew( );

transform-origin: n n ; 변형의 기준점 변경

transform-origin:left 0;
transform-origin:100% 0;
transform-origin:left top;
transform-origin:left 100%;
transform-origin:122px 150px;
transform-origin:100% 0;
