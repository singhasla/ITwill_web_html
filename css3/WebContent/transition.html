transition - 단순한 애니메이션 효과

  • transition-duration (필수) 전체 진행 시간
  • transition-property 효과를 줄 속성명
  • transition-delay 지연 시간
  • transition-timing-function 가속감속 등
    1. ease (기본값 ) = 느리게 시작하고 빨라지다가 천천히 종료하는 효과
    2. ease-in = 느리게 시작하는 효과
    3. ease-out = 느리게 끝나는 효과
    4. ease-in-out = 느리게 시작하고 느리게 끝나는효과
    5. linear = 동일한 속도
    6. cubic-bezier(n,n,n,n) = cubic-bezier 함수값으로 적용
      https://cubic-bezier.com/
