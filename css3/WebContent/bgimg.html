background : 배경

  • background-color: rgb(n,n,n) or #nnnnnn or color명; 배경 색상
  • background-image: url("이미지경로"); 배경 이미지
  • background-repeat: ; 배경 이미지 반복 지정
    1. repeat = 기본값, 양사방 반복
    2. repeat-x = 가로로만 반복
    3. repeat-y = 세로로만 반복
    4. no-repeat = 반복 안함(한번만 보이기)
  • background-size: ; 배경 이미지 크기 지정
    1. contain = 이미지 하나를 온전히 보이게 설정
    2. cover = 이미지 하나로 영역 전체를 덮이게 설정
    3. n n = 가로값 세로값의 순서로 변경(단위는 px,em,% ...)
  • background-position: ; 배경 이미지 위치 지정
    1. l/c/r t/c/b = 수평 수직 순서로 방향값을 조합
    2. n n = 가로값 세로값의 순서로 지정 (단위는 px,em,% ...)
  • background-attachment: ; 배경 이미지 스크롤여부 지정
    1. scroll
    2. fixed
    3. local
