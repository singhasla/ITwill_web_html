margin - 요소의 외부여백

margin값의 갯수별 적용

  • margin: n; 전방향
  • margin: n n; 상하, 좌우
  • margin: n n n; 상, 좌우, 하
  • margin: n n n n; 상, 우, 하, 좌

margin: n; 전방향

div01 / margin: 30px;
div02 / margin: 10%;
div03 / margin: 2em;

margin: n n; 상하, 좌우가 서로 동일

div01 / margin: 20px 5% ;
div02 / margin: 2em 7% ;
div03 / margin: 50px auto;
요소 자체를 정중앙에 위치하려면 좌우 margin값을 동일하게 주면 됨.
따라서 좌우 margin을 auto로 주면 양쪽이 균등분할 되면서 수평 중앙에 위치하게 됨.
