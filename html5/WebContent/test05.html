html5에서 추가된 <input type>의 속성값들

html5에서 모바일환경에 알맞는 폼요소들이 추가됨
input type의 값에 따라 형태, 종류가 달라짐

input type="time" :사용자의 지역을 기준으로 시간(시,분)을 지정

input type="date" :사용자의 지역을 기준으로 날짜(연,월,일)을 지정


<- min="2000-01-01" 적용

input type="datetime-local" :사용자의 지역을 기준으로 날짜와 시간(연,월,일,시,분)을 지정

input type="week" :사용자의 지역을 기준으로 날짜(연,주) 지정

input type="month" :사용자의 지역을 기준으로 날짜(연,월) 지정

input type="number" :숫자값을 지정

min속성으로 입력값의 최소치를,
max속성으로 입력값의 최대치를,
step속성으로 입력값의 간격을 지정할 수 있다.

  1. 개(min="0")
  2. 개(max="100")
  3. 개(단위수량, step="15")

input type="range" :범위 내의 값을 지정

input type="tel" :연락처(전화번호 형식).
모바일 브라우저에서 가상키패드로 숫자 입력

input type="email" 연락처(이메일 형식).
모바일 브라우저에서 가상키패드가 @ 등을 지원할 수 있게 바뀐다.
브라우저에서 알맞은 형식의 값인지 체크해줌.

input type="url" :연락처(웹주소 형식).
모바일 브라우저에서 가상키패드가 https:// 등을 지원할 수 있게 바뀐다.
브라우저에서 알맞은 형식의 값인지 체크해줌.

input type="color" :색상값을 지정

html5에서 추가된 <input>의 속성들

autofocus="autofocus" :커서가 자동으로 미리 위치

autocomplete="on/off" :자동완성기능 on/off(기본값은 on)

placeholder="텍스트" :입력예시나 설명문을 넣어줌


required="required" :필수입력 요소로 지정
브라우저에서 입력데이터의 유무를 체크함.

readonly="readonly" :읽기만 가능한 요소로 지정

당신은 입니다.

disabled="disabled" :사용불가 요소로 지정

당신은 입니다.
