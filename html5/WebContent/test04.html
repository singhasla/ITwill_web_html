form

form은 사용자와 서버를 연결하는 페이지 부분을 구성한다.
사용자가 정보를 서버에 전달할 때 이용하는 다양한 형태의 폼 요소로 이루어진다.
- action 속성 : 정보를 전달 받을 서버측 파일을 지정.
- method 속성 : 정보 전달방식을 지정한다.(일반적으로 속성값은 post)



폼요소

<input type="text"> 한 줄 글상자(주관식)

input상자에 대한 설명 문구를 label태그로 표현한다.
그리고 [label태그의 속성의 값]과 [input태그의 속성의 값]을 일치시켜주면 서로 연결된다.



  • size="n" 상자의 길이
  • maxlength="n" 입력데이터의 최대길이
  • minlength="n" 입력데이터의 최소길이

<input type="password"> 암호 글상자(주관식)


<textarea> ... </textarea> 큰 글상자(주관식)

rows속성으로 상자의 줄을,
cols속성으로 가로 길이를 지정한다.

<input type="radio"> 단일 선택용 라디오버튼(객관식)

단일 선택이 되게 하려면, 같은 질문하의 라디오버튼 항목들끼리는 name속성의 값을 서로 일치시켜준다. 초기값으로 선택되어질 항목에는 checked="checked"속성을 넣어준다.

<input type="checkbox"> 다중 선택용 체크박스(객관식)

코로나19 특별지원금 수령방법으로 선호하는 것은? (중복 선택 가능)



<input type="file"> 파일 전송용

<input type="search"> 검색용 (내용삭제표시 기능)

<input type="hidden"> 사용자에게는 보이지 않는 데이터 전송용

<input type="submit/reset/button"> 제출/취소/일반버튼

value속성의 값으로 버튼의 눈에 보이는 글자를 지정할 수 있다.

<button> ... </button>

시작과 종료태그 사이에 버튼의 글자를 넣어주고,
기본 기능은 전송(submit)이다.
취소 기능은 type="reset"
일반 버튼은 type="button"을 준다

<input type="image"> 전송버튼 역할의 이미지

src속성으로 이미지의 경로를 지정함.
width속성으로 이미지의 가로 크기를,
height속성으로 이미지의 세로 크기를 조절할 수 있다.
