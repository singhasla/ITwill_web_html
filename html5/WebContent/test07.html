내부 링크 걸기

페이지가 길어질 때 (스크롤바 생성시) 해당 페이지 내에서 위로 또는 아래로 특정 영역으로 직접 찾아가는 링크
(주소값의 #은 해당 페이지의 최상단을 의미)
특정 영역의 태그에 id(id는 유일한 값)를 지정하고 그 id값을 주소로 활용한다.
[다른 페이지의 특정 영역으로 이동하는 링크] 1. 해당 페이지의 특정 영역 태그에 id를 지정하고
2. 그 id값을 주소로 활용한다. (예시 > 해당페이지명.html#id값)
